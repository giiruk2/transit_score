# 📐 접근성 점수 산출 알고리즘

## 개요

TransitScore의 핵심 기능은 대중교통 경로 데이터를 **0~100점의 단일 접근성 점수**로 변환하는 것입니다.
이 문서에서는 점수 산출에 사용되는 정규화 공식과 가중치 체계를 상세히 설명합니다.

> 📁 구현 파일: `backend/src/services/score.ts`

---

## 입력 데이터

ODsay 대중교통 길찾기 API (또는 Fallback 추정 로직)로부터 다음 4가지 원본 지표를 수신합니다.

| 지표 | 변수명 | 단위 | 설명 |
|------|--------|------|------|
| 총 이동시간 | `totalTimeMin` | 분 | 출발지→도착지 전체 소요시간 |
| 환승 횟수 | `transferCount` | 회 | 버스↔지하철 간 환승 수 |
| 도보 거리 | `walkDistanceM` | 미터 | 정류장까지/에서 걸어야 하는 총 거리 |
| 대기 시간 | `waitTimeMin` | 분 | 배차 대기시간 |

---

## 정규화 공식 (0~1 스케일)

각 지표를 0~1 사이의 점수로 변환합니다. **1에 가까울수록 접근성이 좋음**을 의미합니다.

### 1️⃣ 시간 점수 (`s_time`)

```
s_time = 1 - (totalTimeMin - T_best) / (T_worst - T_best)
```

| 파라미터 | 값 | 의미 |
|----------|-----|------|
| `T_best` | 20분 | 최상의 이동시간 (20분 이하는 만점) |
| `T_worst` | 120분 | 최악의 이동시간 (120분 이상은 0점) |

**예시:** 이동시간 50분 → `s_time = 1 - (50-20)/(120-20) = 0.70`

### 2️⃣ 환승 점수 (`s_transfer`)

계단형(Stepped) 패널티를 적용합니다.

| 환승 횟수 | 점수 |
|-----------|------|
| 0회 (직행) | **1.00** |
| 1회 | **0.75** |
| 2회 | **0.50** |
| 3회 | **0.25** |
| 4회 이상 | **0.10** |

### 3️⃣ 도보 점수 (`s_walk`)

```
s_walk = 1 - (walkDistanceM / D_max)
```

| 파라미터 | 값 | 의미 |
|----------|-----|------|
| `D_max` | 1,200m | 최대 허용 도보 거리 (초과 시 0점) |

**예시:** 도보 600m → `s_walk = 1 - (600/1200) = 0.50`

### 4️⃣ 대기 점수 (`s_wait`)

```
s_wait = 1 - (waitTimeMin / W_max)
```

| 파라미터 | 값 | 의미 |
|----------|-----|------|
| `W_max` | 20분 | 최대 허용 대기시간 (초과 시 0점) |

---

## 가중합 산출 (최종 점수)

정규화된 5개 항목에 가중치를 곱한 뒤 합산하여 100점 만점으로 환산합니다.

```
최종점수 = 100 × (0.45 × s_time + 0.20 × s_transfer + 0.15 × s_walk + 0.10 × s_wait + 0.10 × s_access)
```

### 가중치 배분 근거

| 항목 | 가중치 | 근거 |
|------|--------|------|
| **이동시간** | **45%** | 관광객이 가장 민감하게 체감하는 핵심 요소 |
| **환승** | **20%** | 짐이 많은 관광객에게 환승은 큰 부담 |
| **도보** | **15%** | 정류장까지 걸어야 하는 거리 역시 접근성에 영향 |
| **대기시간** | **10%** | 배차 간격에 따른 불편함 |
| **무장애 정보** | **10%** | 엘리베이터 유무 등 (현재 기본값 0.5 = 중립, 추후 TourAPI 연동 예정) |

---

## 계산 예시

> 부산역 → 태종대유원지 (ODsay 실제 데이터 기준)

| 지표 | 원본값 | 정규화 점수 |
|------|--------|-------------|
| 이동시간 | 45분 | `1 - (45-20)/(120-20)` = **0.75** |
| 환승 | 1회 | **0.75** |
| 도보 | 450m | `1 - (450/1200)` = **0.63** |
| 대기 | 5분 | `1 - (5/20)` = **0.75** |
| 무장애 | 0.5 (기본) | **0.50** |

```
최종점수 = 100 × (0.45×0.75 + 0.20×0.75 + 0.15×0.63 + 0.10×0.75 + 0.10×0.50)
        = 100 × (0.3375 + 0.15 + 0.0945 + 0.075 + 0.05)
        = 100 × 0.707
        = 70.7점
```

## 점수 해석 가이드

| 점수 구간 | 등급 | 의미 |
|-----------|------|------|
| 80~100 | 🟢 우수 | 직행 또는 1회 환승, 30분 이내 도착 |
| 60~79 | 🟡 양호 | 1~2회 환승, 1시간 이내 도착 |
| 40~59 | 🟠 보통 | 2회 이상 환승, 도보 거리가 김 |
| 0~39 | 🔴 미흡 | 2시간 가까이 소요, 접근성이 매우 낮음 |
