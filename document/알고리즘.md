# 📐 대중교통 접근성 분석 알고리즘

본 문서는 TransitScore 가 대중교통 경로 데이터를 바탕으로 어떻게 '접근성 점수'를 산출하는지에 대한 모든 세부 로직을 담고 있습니다.

---

## 1. 개요
TransitScore 의 알고리즘은 **이동 시간, 환승 횟수, 도보 거리, 대기 시간**의 4개 핵심 지표를 정규화(Normalization) 한 뒤, 각각의 지표에 가중치를 곱하여 합산(Weighted Sum) 하는 방식을 취합니다. 

최종 결과값은 **0점(접근성 매우 낮음)에서 100점(접근성 매우 우수)** 사이의 수치로 반환됩니다.

---

## 2. 지표별 정규화 공식 (0 ~ 1.0)

모든 데이터는 먼저 0과 1 사이의 점수로 변환됩니다. (1.0에 가까울수록 접근성이 좋음을 의미함)

### ① 시간 점수 ($S_{time}$)
- **기준**: 최소 20분($T_{min}$), 최대 120분($T_{max}$)
- **공식**: $1 - \frac{소요시간 - 20}{120 - 20}$
- **설명**: 20분 이내 도착 시 만점(1.0), 120분 이상 소요 시 0점이며, 그 사이 시간은 선형적으로 점수가 깎입니다.

### ② 환승 점수 ($S_{transfer}$)
- **기준**: 계단식 페널티 적용
- **공식**:
  - 0회(직행): 1.0
  - 1회: 0.75
  - 2회: 0.5
  - 3회: 0.25
  - 4회 이상: 0.1
- **설명**: 환승은 관광객의 피로도를 높이는 핵심 요인이므로 이동 시간 다음으로 큰 가중치를 가집니다.

### ③ 도보 점수 ($S_{walk}$)
- **기준**: 최대 허용 도보 거리 1,200m ($D_{max}$)
- **공식**: $1 - \frac{도보거리}{1200}$
- **설명**: 정류장에서 목적지까지 1.2km를 넘어가면 도보 점수는 0점이 됩니다.

### ④ 대기 점수 ($S_{wait}$)
- **기준**: 최대 대기 시간 20분 ($W_{max}$)
- **공식**: $1 - \frac{대기시간}{20}$
- **설명**: 배차 간격에 따른 불확실성을 반영합니다.

---

## 3. 종합 점수 산출 (가중합)

정규화된 각 점수에 정해진 가중치를 곱하여 합산한 뒤 100을 곱해 최종 점수를 만듭니다.

| 평가 항목 | 가중치 | 근거 |
| :--- | :--- | :--- |
| **이동 시간** | **45%** | 사용자가 느끼는 가장 직접적인 접근성 지표 |
| **환승 횟수** | **20%** | 환승 시 발생하는 번거로움과 피로도 반영 |
| **도보 거리** | **15%** | 정류장 주변 접근 편리성 |
| **대기 시간** | **10%** | 실시간 배차 및 대기 불편함 |
| **무장애 요소** | **10%** | 현재 기본값(0.5) 적용 (추후 데이터 보강 예정) |

### 🚩 최종 수식
$$\text{Final Score} = 100 \times (0.45 \cdot S_{time} + 0.20 \cdot S_{transfer} + 0.15 \cdot S_{walk} + 0.10 \cdot S_{wait} + 0.10 \cdot 0.5)$$

---

## 4. 특수 상황 처리 (Fallback 로직)

API 한도 초과(Rate Limit)나 경로 탐색 실패 시, 서비스 중단을 막기 위해 **자체 추정 알고리즘**이 가동됩니다.

- **거리 계산**: 하버사인(Haversine) 공식을 이용한 두 지점 간 직선 거리 산출
- **시간 추정**: 대중교통 평균 이동 속도인 **15km/h**를 기준 속도로 적용
- **도보 추정**: 직선 거리의 **15%**를 도보 이동량으로 가제(Assumption)
- **UI 알림**: 정확한 API 데이터가 아님을 나타내는 **fallback 배지**를 지도 상세 패널에 표시

---

## 5. 점수 해석표
- **80 ~ 100**: 🟢 **매우 우수** (30분 내외, 직행 또는 원활한 환승)
- **60 ~ 80**: 🟡 **우수/양호** (1시간 내외, 대중교통 이용 권장)
- **40 ~ 60**: 🟠 **보통** (환승 횟수가 많거나 도보가 다소 긺)
- **0 ~ 40**: 🔴 **미흡** (대중교통 이용보다 택시나 자차 이용 권장)
